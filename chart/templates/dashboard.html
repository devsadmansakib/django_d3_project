About This Project

This project showcases how to create a full-stack Django application with both web and API functionality:

  • **D3.js Visualization**: A chart rendered with D3.js, accessible after user login.
  • **Token-Based Authentication**: Secure REST API endpoints, allowing only authorized access.
  • **Bootstrap UI**: A responsive and modern UI using Bootstrap.
